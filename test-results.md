## 테스트 시나리오
1. **할 일 추가**
   - 제목만 입력하여 추가
   - 제목과 설명 모두 입력하여 추가
   - 빈 제목으로 시도 (실패해야 함)

2. **할 일 완료 체크**
   - 체크박스 클릭하여 완료 상태 토글
   - 완료된 항목은 회색 배경과 취소선 스타일 적용

3. **할 일 삭제**
   - X 버튼 클릭하여 삭제
   - 삭제 후 목록에서 즉시 제거
