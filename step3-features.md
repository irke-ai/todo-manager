# Step 3 - 고급 기능 추가 완료

## 구현된 기능

### 1. 우선순위 설정 ✅
- TodoForm에 우선순위 선택 dropdown 추가
- 높음/중간/낮음 3단계 선택 가능
- 기본값: 중간

### 2. 마감일 설정 ✅
- datetime-local input으로 날짜와 시간 선택

- **우선순위 필터**: 전체 / 높음 / 중간 / 낮음
- 실시간 필터 적용

### 4. 정렬 기능 ✅
- **정렬 기준**: 생성일 / 마감일 / 우선순위
- **정렬 순서**: 오름차순 / 내림차순 토글
- 화살표 버튼으로 순서 변경
